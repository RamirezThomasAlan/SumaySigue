Suma y Sigue

¿Cómo jugar?

  • Desliza con el dedo (en celulares) o arrastra con el mouse (en PC) para mover los bloques.
  • Cuando dos bloques del mismo número se tocan, se fusionan en uno con la suma de ambos.
  • Después de cada movimiento, aparecerá un nuevo bloque de valor 2 o 4.
  • El objetivo es crear un bloque con el número 2048.
  • ¡Obtienes puntos por cada fusión que realices!
